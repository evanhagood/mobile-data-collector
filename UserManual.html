Field Day Wildlife Data Collection App

Introduction

The Field Day project is an innovative Wildlife Data Collection and management Tool designed to address field researchers' challenges, particularly those in biology and ecology.

Key Features:
  • Progressive Web App (PWA) for mobile devices
  • Web application for larger screens
  • Offline functionality with local database caching
  • 2-factor Google authentication
  • Population data collection using capture mark-recapture techniques

Getting Started

Copy

Logging In

  1. Access the PWA at https://asu-field-day-pwa.web.app/
  2. Tap the Login button
  3. Use your ASU.edu credentials for Google OAuth 2.0 authentication
  4. Complete 2-factor authentication if prompted
  5. Ensure App Mode: Live is selected

Navigation

Access the main menu from the top-left corner for:

  • Collect Field Data: Begin a new session
  • Session History: View past sessions
  • About Us: Project team information

Data Collection Process

Copy

Starting a New Session

  1. Enter handler initials (2-3 characters required)
  2. Select your project
  3. Choose site and array

Recording Captures

Select the appropriate form for:

  • Arthropod
  • Amphibian
  • Lizard
  • Mammal
  • Snake

Mammal Entry Form

  • Record species details
  • Enter measurements
  • Add observations

Snake Entry Form

  • Document snake species
  • Record measurements
  • Note observations

Trap Status

Indicate trap status as:

  • Open
  • Checked
  • Checked and closed

Best Practices

  • Always verify data before final submission
  • Enter comments for any unusual observations
  • Ensure proper trap status documentation
  • Complete all required fields

Technical Notes

  • Offline data collection available through local caching
  • Regular synchronization recommended
  • Keep Google Authenticator app updated for 2FA
